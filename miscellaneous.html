HTML Entities

Entities are used to display reserved characters:

  • &lt; displays: <
  • &gt; displays: >
  • &amp; displays: &
  • &copy; displays: ©
  • &nbsp; displays:   (non-breaking space)

HTML Quotation Tag

The <blockquote> tag is used for long quotations:

"This is a sample block quotation from an external source."

The <q> tag is used for short inline quotations: This is an inline quote.

Obsolete HTML Tags

Some HTML tags are obsolete and not recommended for use:

  • <font> (use CSS for styling text instead)
  • <center> (use CSS text-align instead)
  • <marquee> (no longer supported)
  • <u> (use CSS text-decoration instead)

This text uses the obsolete <font> tag.

Character Sets

The character set defines how characters are encoded. UTF-8 is the most common and supports most languages:

<meta charset="UTF-8">
